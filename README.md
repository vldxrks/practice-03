# Практична робота №3: Цикли (Варіант 1)
# Рукасов Владислав Ік-42
## Опис проєкту
Цей репозиторій містить рішення задач на обробку масивів з використанням різних циклів та алгоритмічних патернів (Two Pointers, In-place модифікація).

## Аналіз складності алгоритмів:
1. **Maximum Subarray Sum (Kadane's Algorithm)**
   - **Часова складність:** O(n) – проходимо масив один раз.
   - **Просторова складність:** O(1) – додаткова пам'ять не використовується, лише дві змінні.
2. **Rotate Array**
   - **Часова складність:** O(n) – елементи реверсуються 3 рази.
   - **Просторова складність:** O(1) – реверс відбувається in-place (без створення нових масивів).
3. **Move Zeroes**
   - **Часова складність:** O(n) – один цикл `for`.
   - **Просторова складність:** O(1) – модифікація масиву in-place.
4. **Two Sum II (Sorted Array)**
   - **Часова складність:** O(n) – використовується патерн Two Pointers (два вказівники наближаються один до одного).
   - **Просторова складність:** O(1).
5. **Container With Most Water**
   - **Часова складність:** O(n) – Two Pointers.
   - **Просторова складність:** O(1).
